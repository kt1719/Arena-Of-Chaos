# Analysis Framework: Experience Map

Use this to map the emotional arc of a game or game session.

## Step 1: Identify the Experience Timeline
Break the game into phases (first 5 minutes, early game, mid game, late game, endgame) or specific sequences.

## Step 2: Map Emotional Triggers Per Phase
For each phase, identify which emotional triggers are active:

| Phase | Learning | Challenge | Social | Acquisition | Narrative | Music/Audio | Spectacle | Beauty | Environment | Flow State? |
|-------|----------|-----------|--------|-------------|-----------|-------------|-----------|--------|-------------|-------------|
| | | | | | | | | | | |

## Step 3: Map Human Value Shifts
For each phase, what human values are shifting?
- [victory/defeat], [life/death], [knowledge/ignorance], [wealth/poverty]
- [together/alone], [high status/low status], [danger/safety]
- How large are the shifts? How important are the values?

## Step 4: Plot the Pacing Curve
Rate emotional intensity (1-10) across the timeline. Compare to the classic curve:
```
Hook → Rising Action → Climax → Denouement
```
- Are there flow gaps (boring stretches with no decisions)?
- Are there overflow points (too many decisions, player is overwhelmed)?
- Is there emotional variation (changes in valence, not just intensity)?

## Step 5: Check for Anti-Patterns
- [ ] **Flat line**: No variation in intensity → monotonous
- [ ] **All peaks**: Constant high intensity → exhausting, nothing feels special
- [ ] **Antagonistic emotions**: Are any feelings destroying each other?
- [ ] **Missing triggers**: Is the game relying on only 1-2 emotional triggers?
- [ ] **Flow breaks**: Where does the player get pulled out of concentration?
- [ ] **Fiction-mechanics mismatch**: Does the fiction label the arousal correctly for immersion?

## Step 6: Identify the Worst Experience
Apply the **fallacy of vision antidote**: instead of the best possible playthrough, map the *worst*. What does a frustrating, boring, or confusing session look like?

## Output Format
- **Emotional arc summary**: One paragraph describing the intended experience
- **Strongest moments**: Where the design shines
- **Weakest moments**: Where emotion sags or breaks
- **Missing triggers**: Emotional triggers not being used that could enhance the experience
- **Recommendations**: Specific changes to improve the emotional arc
