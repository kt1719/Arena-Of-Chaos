# Analysis Framework: Value Curve

Use this for market positioning and competitive analysis.

## Step 1: Identify Market Values
List every distinct type of value the game offers. Examples:
- Hardcore tactical combat, creative building, emergent narratives
- Atmospheric exploration, competitive multiplayer, cooperative play
- Specific fiction appeals (cyberpunk, fantasy, horror, humor)
- Unique mechanics (time manipulation, physics sandbox, procedural generation)

## Step 2: Rate Your Game
For each value, rate how well your game delivers it (1-10).

## Step 3: Identify Competitors
List 3-5 games that compete for the same players. Rate them on the same values.

## Step 4: Plot the Curves
```
Value          | Your Game | Comp A | Comp B | Comp C
---------------|-----------|--------|--------|-------
[value 1]      |           |        |        |
[value 2]      |           |        |        |
[value 3]      |           |        |        |
...            |           |        |        |
```

## Step 5: Analyze

### Superior Values (you > all competitors)
These are your selling points. Players who want these values must come to you.

### Unique Values (only you offer them)
**The greatest victory.** No competition at all. BioShock's art deco world. Garry's Mod's wacky creativity.

### Inferior Values (competitors > you)
Not selling points. Players who want these values have better options elsewhere. **Don't invest more here unless you can become superior.**

### Absent Values (you don't offer, competitors do)
Intentional absences are fine. Not every game needs multiplayer or a story. But unintentional absences may represent missed opportunities.

## Step 6: Shape Check
- **Tall spike** (1-2 values pushed high) → Focused, viable for small teams. Good.
- **Moderate peaks** (3-4 superior values) → Viable for medium teams. Good.
- **Flat stump** (many values, none superior) → **DANGER.** Mediocre at everything, best at nothing. No reason for anyone to play this over competitors.

## Step 7: Resource Alignment
Does your team's size and budget match your value curve ambition?
- Small team → must focus on 1-2 values maximum
- Large team → can attempt broader curve but still needs clear peaks
- **"Better to be the best at one thing than mediocre at ten things"**

## Output Format
- **Value curve table**: Your game vs. competitors
- **Shape diagnosis**: Spike, peaks, or stump?
- **Unique advantages**: Values with no competition
- **Wasted effort**: Values where you're inferior and shouldn't invest more
- **Recommendations**: Where to focus development resources
